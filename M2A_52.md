### Desabilitando error no standard em funcões globais
Para remover esse tipo de error no standard basta adicionar o seguinte propriedade no arquivo `package.json`.

```
"standard": {
    "globals": [
      "alert",
      "test",
      "it",
      "describe"
    ]
  }
```
