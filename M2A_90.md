### Instalando

```
yarn add --dev raw-loader
```

### Configurando
Configuramos os arquivos que utilizaram o raw-loader e habilitamos o plugin.

```
module.exports = {
  entry: resolve('index.js'),

  output: {
    path: join(__dirname, '..', 'example-loaders/dist'),
    filename: 'bundle.js'
  },

  module: {
    rules: [{
      test: /\.txt$/,
      use: 'raw-loader'
    }]
  }
}
```
